# PawPoint: Veterinary Appointment System

A web-based application to simplify appointment scheduling, pet records, and clinic operations for veterinary clinics. The system supports multiple user roles (Super Admin, Admin/Receptionist, Veterinarian, Pet Owner) and provides appointment booking, treatment record tracking, and clinic management features.

## Key Features

- User Authentication (Super Admin, Admin/Receptionist, Veterinarian, Pet Owner)
- Appointment Scheduling with status updates (booked, confirmed, in-progress, completed, cancelled)
- Pet & Owner Management
- Veterinarian Management and Availability Tracking
- Clinic Data Management (services, locations, hours)
- Treatment Record Tracking and History
- Responsive Web Interface (React frontend + Flask backend)
